Features

Used for self tracking - tracking habits, activities, other life parameters etc. User can have multiple trackers. User can log to one more tracker at any time. System will track progress over time and shows graphs trend lines etc.
  • User login and Sign Up
  • Once logged in, the user is able to create trackers which can be either numeric or multiple choice type.
  • The dashboard shows a list of all trackers, most recent value and last updated time.
  • Upon going to the tracker screen, a line chart for numeric and a scatter plot for multiple choice type tracker is shown, taking time on the x axis.
  • User can update a tracker and their previous logs at any time.

User Guide

  • 1. User Sign Up

    Click on home and click on sign up button. Enter username and password and click on sign up. Once your profile is created, you will be redirected to the login page, where you can enter your details and login.

  • 2. Dashboard

    Once you are logged in, you can see the dashboard. It usually contains the list of trackers that you have created. Here you can navigate to different trackers, update or delete a particular tracker and also update your profile.

  • 3. Add Tracker

    By clicking on add tracker button, you will be redirected to a form. Here, you can enter the details about the tracker. Depending on the type of tracker, you will be asked to enter additional information.

  • 4. Logging Values

    Action of recording values into a tracker is called logging. Once you have created a tracker, click on it to view it's details and record values.

  • 5. Tracker Screen

    Each tracker has details about itself. It also has trendlines plotted with respect to time, and recent logs.

  • 6. All your data

    In tracker screen, click on "see all logs" to see all the data recorded in a particular tracker. You can update or delete these logs if you wish.
