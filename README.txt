The following python libraries must be installed to run the application.
Flask
Flask-SQLAlchemy
flask-restful
flask-cors
matplotlib
***********************************************************
To run the application execute the following commands on a linux terminal:
1. Use cd to navigate to the root folder
2. Set up a virtual python environment using the command "python3 -m venv .env-name" (substitute env-name with required environment name)
3. run the command "pip3 install -r requirements.txt" to install the mentioned python libraries
4. run the command "source .env-name/bin/activate" to activate the virtual environment
5. run "python app.py" to start the application
6. paste the url  "http://127.0.0.1:5000/"( - default flask port) onto a web browser like chrome, safari, firefox.
